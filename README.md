# DenDen

🔗 **Canlı site:** [azizenesozturk.github.io/denden-site](https://azizenesozturk.github.io/denden-site/)

Kişisel portfolyo ve dağcılık/doğa keşfi günlüğü sitesi.

## Hikayesi

Bir dağcılık kulübü için web sitesi yapmak istiyordum, ama onlara hazırlamadan önce kendimi denemek istedim. Bu fikirle yola çıkıp önce kendime bir site yaptım. İleride aynı altyapıyla kulüp için de bir site hazırlamayı planlıyorum.

Sunucuyla uğraşmamak için sadece HTML, CSS ve JavaScript ile (statik/sunucusuz mimari) yazdım. Bu yaklaşım yeterli geldiği için kulüp sitesini de aynı şekilde kodlayacağım. İçerik (yazı ve fotoğraf) eklemeyi kolaylaştırmak için ileride kendime özel küçük scriptler de yazmayı planlıyorum.

## Bölümler

- **Faaliyetler** — Geçmiş rotalarım ve yaklaşan faaliyetlerim
- **Blog** — Faaliyet anıları ve yazılarım
- **Duyuru** — Güncel duyurular
- **Çalışmalarım** — Yazılım projelerim
- **Oyun** — Basit bir tarayıcı oyunu
- **Bağış** — Destek olmak isteyenler için (yakında)

## Teknoloji

- HTML, CSS (Tailwind CDN), Vanilla JavaScript
- Veri yönetimi için JSON dosyaları
- Sunucusuz mimari — statik site olarak GitHub Pages üzerinde barındırılıyor

## İletişim

Sorularınız veya birlikte doğa keşfine çıkmak isterseniz, sitedeki İletişim sayfasından bana ulaşabilirsiniz.
